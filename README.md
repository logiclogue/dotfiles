# @logiclogue dotfiles

## Configuration

`config.json`:
```
{
    "desktop": true
}
```

## Devices

- `laptop`
- `desktop`
- `work`
- `mobile`
- `server`

## Requirements

- `npm`
- `node`
- `git`
- `make`

## Installation

```
git clone https://github.com/logiclogue/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
make
```
